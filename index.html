Minha Calculadora: Uma Calculadora Simples.

Este projeto consiste em uma calculadora simples desenvolvida com HTML, CSS e JavaScript. A proposta foi criar uma aplicação web funcional que permita ao usuário realizar operações matemáticas básicas de forma prática e intuitiva.

A estrutura da calculadora foi construída com HTML, organizando os botões de números e operadores em uma interface limpa e de fácil compreensão. Para o design visual, utilizei CSS, aplicando estilos que tornam o layout responsivo, moderno e agradável aos olhos, com cores suaves e botões interativos que respondem ao clique do usuário.

A lógica da calculadora foi implementada em JavaScript, permitindo a realização de operações como adição, subtração, multiplicação e divisão. Além disso, o script lida com situações como múltiplos cliques em operadores, limpeza da tela, e o uso do ponto decimal. Todo o funcionamento é feito no lado do cliente, sem a necessidade de conexão com servidores ou bancos de dados.

Este projeto me ajudou a praticar a manipulação do DOM, eventos em JavaScript, e aplicar conceitos importantes de lógica de programação. Também me permitiu trabalhar na organização do código e melhorar minhas habilidades em front-end.

Meu Site de Sistemas: Um Site Pessoal.

Este projeto é um site pessoal que desenvolvi com o objetivo de apresentar meus conhecimentos, projetos e habilidades na área de desenvolvimento de sistemas. A aplicação foi construída utilizando React para a interface do usuário, Node.js no back-end e MySQL como banco de dados, permitindo uma estrutura sólida, moderna e escalável.

A interface foi desenvolvida com React, proporcionando uma navegação fluida, componentização eficiente e uma experiência dinâmica para o usuário. No back-end, utilizei Node.js com Express para gerenciar as rotas e processar as requisições, garantindo um bom desempenho na comunicação entre o cliente e o servidor. Para armazenamento de dados, como mensagens de contato ou informações de projetos, utilizei o banco de dados MySQL, com consultas otimizadas e seguras.

Além disso, implementei boas práticas de desenvolvimento como responsividade, organização de código, validações de entrada e separação de responsabilidades entre front-end e back-end. Este projeto é tanto uma ferramenta de apresentação profissional quanto uma base para futuras evoluções, como integração com APIs externas ou um painel administrativo.
